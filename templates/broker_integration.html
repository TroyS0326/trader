Broker Integration FAQs

Which brokers does XeanVI currently support?
XeanVI is currently designed around Alpaca API support for paper trading and broker-connected automation workflows. Users should confirm supported account types, permissions, and availability before relying on any live workflow.
Can XeanVI withdraw or transfer my funds?
XeanVI is designed for trading workflow automation and broker connectivity, not fund transfers. Users should always review the exact permissions shown by the broker during authorization and disconnect access if they no longer want the integration active.
Do I need to leave my computer running for supported API workflows?
XeanVI is designed for web-based workflow access, so users can manage settings from the dashboard rather than relying only on a local desktop session. You should still monitor account activity, broker status, and automation settings when using live tools.
Should I test in paper trading before live trading?
Yes. Paper trading is strongly recommended before enabling any live broker-connected workflow. It helps you review setup behavior, order settings, playbook rules, and risk controls without using real capital.
