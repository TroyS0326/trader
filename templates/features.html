XeanVI Features FAQs

Do I need to know how to code to use XeanVI?
No coding experience is required. XeanVI is designed around a visual dashboard and Trading Playbook where users can define rules, risk settings, and workflow preferences without writing code.
Can I test features before using live trading tools?
Yes. XeanVI is designed to support paper trading so users can test workflows, review signals, and adjust playbook rules before deciding whether to connect supported live broker tools.
Does XeanVI provide financial advice?
No. XeanVI provides software tools for trading workflow automation, market review, paper trading, and broker connectivity. It does not provide financial advice, investment recommendations, or guaranteed trading results.
