# Viral First Asset Batch

## Purpose

This document contains the first production-ready Viral asset prompt batch for XeanVI blog images and social media posts.

These prompts are designed to test Viral's consistency across the highest-value XeanVI content pillars before adding her to any live site or app component.

---

## Usage rules

1. Generate these as still images first.
2. Do not publish all outputs automatically.
3. Reject any output that looks sexualized, unserious, too cartoonish, too hype-driven, or inconsistent with the Viral charter.
4. Do not add embedded text inside generated images.
5. Use captions, alt text, and metadata outside the image file.
6. Do not use images that show fake profit numbers, account balances, real broker logos, or recognizable real people.

Required source documents:

```txt
docs/brand/viral-character-charter.md
docs/brand/viral-prompt-system.md
```

---

## Global negative prompt

Use this for every image generator that supports negative prompts.

```txt
sexualized, seductive pose, revealing outfit, lingerie, glamour model, luxury car, mansion, money pile, cash, fake account balance, profit guarantee, stock pick, buy signal, sell signal, price target, casino, gambling, cartoon, anime, mascot, childish, exaggerated expression, chaotic chart, messy dashboard, blurry, low resolution, text, words, logo, watermark, real celebrity likeness, real person likeness
```

## Batch review process

After generating the first 10 images:

1. Choose the 3 strongest images.
2. Reject anything that looks like a generic influencer account.
3. Reject anything that looks too sexy, childish, cartoonish, or fake-rich.
4. Prefer images where Viral looks serious, calm, and credible.
5. Keep a folder of approved references before generating video.
6. Do not move to video until a stable still-image identity is selected.

---

## Hard stop rule

Do not create video assets until still-image consistency is approved.

If Viral's face, age, style, or professionalism changes too much across images, stop and refine the image prompt system before moving forward.
